# Javascript - banking
- 가상 은행거래 사이트
- Jsonwebtoken, Bcypt, MongoDB를 이용하여 회원가입, 로그인, 이체내역(송금/대출) 저장
- Heroku에 banking 프로젝트 배포
<hr />

- Open Account(Register), Login, Transfer Money and Loan via MongoDB
- Deploy Banking Project on Heroku
